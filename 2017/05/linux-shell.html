grep

-n 显示行号
-o 只显示匹配的内容
-i 忽略大小写
-q 静默
-v 取反，其它行
-w 单词
-l 过滤成功显示文件路径
-A 显示匹配成功后的几行 <要加参数 n>
-B 显示匹配成功前的几行 <要加参数 n>
-C 显示匹配成功的附近几行 <要加参数 n>
-c 显示匹配成功的数目

注：最好用 egrep
